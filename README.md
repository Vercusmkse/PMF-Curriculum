# PMF Digital Skills Curriculum Blueprint & Operational Hub

This is a web project for the "PMF Digital Skills Curriculum Blueprint & Operational Hub". It's designed to provide a comprehensive overview and operational guide for a 16-week web development training program.

## Technology Stack

*   **HTML5**
*   **Tailwind CSS**
*   **JavaScript (Pure JS)**

## Project Overview

The main entry point for the website is `index.html`. This file lays out the structure and content for the entire single-page application. The website is designed to be a central hub for students and facilitators involved in the Phucuka Mzansi Foundation's digital skills program.

### Key Features

*   **Interactive Syllabus Tracker:** A progress bar and checklist to track completion of the 16-week curriculum.
*   **Operational Roadmap:** A timeline of the bootcamp's operational phases, from recruitment to graduation.
*   **Curriculum Navigator:** A detailed, week-by-week breakdown of the syllabus, with links to learning materials.
*   **FNB App Academy Integration:** A section dedicated to the FNB App Academy, featuring a video library and links to relevant resources.
*   **Facilitator and Support Information:** Details on the instructional team, their responsibilities, and the weekly support schedule.

### Referenced Documents

The curriculum and operational plan are based on the following documents:

*   `Phucuka_Mzansi_Curriculum_Blueprint.pdf`
*   `Software Development .pdf`
*   `pmf tech project.pdf`

## Purpose

The goal of this project is to provide a clear and interactive platform for the Phucuka Mzansi Foundation's web development bootcamp. It aims to equip South African youth with high-impact digital skills and programming capabilities through an entirely online learning experience.
